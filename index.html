Como é a estrutura básica de um HTML?

Um documento HTML é formado por várias tags, que dizem ao navegador o que mostrar e como organizar cada parte da página.

      <html>  Início do documento HTML. Todo o conteúdo da página deve estar dentro dessa tag.
      
      <head> 
      Cabeçalho da página. Contém informações invisíveis ao usuário, como título, codificação de caracteres e links para estilos (CSS).
      </head>

      <body> 
      Corpo da página. Tudo que o usuário vê está aqui: textos, imagens, botões, formulários, vídeos etc.
      </body>

      </html> Fim do documento HTML. Fecha a estrutura iniciada com a tag <html>.

Diferenças entre Listas

A tag <ol> define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética.
A tag <li> é usada para definir cada item da lista.

Exemplo:

Código
<ol>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>
Retorno da página HTML
  1. Coffee
  2. Tea
  3. Milk

A tag <ul> define uma lista não ordenada (com marcadores).

Exemplo:

Código
<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>
Retorno da página HTML
  • Coffee
  • Tea
  • Milk

Diferenças entre Checkbox, Radio e Select

Checkbox: permite selecionar uma ou mais opções ao mesmo tempo.
Radio: permite escolher apenas uma opção entre várias (opções exclusivas).
Select: exibe uma lista suspensa com várias opções, normalmente para escolha única.

Formatos de mídia aceitos no HTML5

As tags utilizadas são:

  • <img> – Para exibir imagens.
  • <video> – Para incorporar vídeos com controles.
  • <track> – Para legendas e faixas de texto em vídeos.
  • <iframe> – Para incorporar outros conteúdos (como vídeos do YouTube ou sites).
